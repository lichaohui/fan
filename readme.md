#### 安装与运行
1. npm install
2. bower install
3. gulp serve

#### 页面列表
* area - 区域(包含标签筛选弹出框)
* bind_phone - 微信登录（绑定手机号）
* busdis - 商圈
* destine - 预定
* financial - 金融服务
* forget_password - 忘记密码
* index - 首页
* init_password - 初始化密码
* login - 登录
* register - 注册
* register_wechat - 微信注册
* search - 搜索
* setting - 设置
* shop - 店源
* user - 个人中心首页
